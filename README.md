# Mind & Progress

아이디어를 구조화하는 마인드맵과 실제 업무 진행률을 한 화면에서 연결하는 웹 서비스의 초기 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

최초 실행 시 관리자 계정이 없으면 API 서버 콘솔에 일회성 임시 비밀번호가 표시됩니다. 초기 계정을 직접 지정하려면 실행 전에 환경변수를 설정합니다.

```powershell
$env:MNP_ADMIN_EMAIL='admin@example.com'
$env:MNP_ADMIN_PASSWORD='8자 이상의 안전한 비밀번호'
npm run dev
```

환경변수는 관리자 계정을 처음 생성할 때만 적용됩니다. 로그인 후 사용자 메뉴의 `비밀번호 변경`에서 비밀번호를 교체해 주세요. 문서, 댓글, 계정 및 변경 이력이 저장되는 `server/data`의 런타임 파일은 Git에 포함되지 않습니다.

## 현재 구현 범위

- 노드 선택, 이동, 연결, 추가 및 삭제
- 노드별 제목, 설명, 상태, 진행률 편집
- 서버 세션 로그인과 편집자/뷰어 권한 검증
- 뷰어의 저장 요청 서버 차단
- 여러 마인드맵 문서 생성, 전환, 이름 변경
- 문서별 서버 자동 저장과 로컬 백업
- 노드를 업무로 전환하고 담당자·마감일 지정
- 체크리스트 완료율과 노드 진행률 자동 동기화
- 칸반에서 업무 상태별 조회와 드래그 이동
- 마감일 기준 업무 타임라인
- 완료율, 기한 초과, 담당자 분배 진행률 대시보드
- 문서별 최대 100단계 실행 취소·다시 실행 (`Ctrl+Z`, `Ctrl+Y`)
- 미니맵, 확대/축소, 전체 보기
- 하위 가지 접기·전체 펼치기와 노드 검색·상태·담당자 필터
- 업무 링크 연결 및 완료 상태 표시
- 문서별 최근 50개 변경 이력 조회와 안전한 이전 버전 복원
- 다른 브라우저의 문서 변경 실시간 감지와 뷰어 자동 반영
- 같은 문서 접속자와 실시간 캔버스 커서 표시
- 문서 버전 충돌 감지와 노드·필드 단위 3방향 자동 병합
- 노드별 실시간 댓글, 답글 스레드, 멘션, 해결 상태와 이모지 반응
- 사용자별 실시간 알림함과 개별·전체 읽음 처리
- 로그인 계정 담당자 배정과 마감 3일 전·당일·기한 초과 개인 알림
- 관리자 전용 편집자 계정 생성·수정·정지·비밀번호 초기화·삭제
- 로그인 사용자의 현재 비밀번호 확인 기반 비밀번호 변경
- 선행·후속 업무 관계, 순환 방지와 차단 병목 표시

## 접속 경로

- 일반 로그인: `http://127.0.0.1:4175/`
- 공개 읽기 전용: `http://127.0.0.1:4175/viewer`

로그인 화면은 계정 정보를 미리 채우지 않습니다. 로그인에 성공한 이메일만 해당 브라우저에 기억하며 비밀번호는 저장하지 않습니다.

공개 읽기 전용 화면은 전용 시스템 계정으로 분리되어 문서와 댓글 조회 및 업무 링크 열기만 허용합니다. 댓글 작성·반응과 문서 변경은 서버에서도 차단합니다. 편집자는 로그인 후 우측 상단 사용자 이름을 눌러 `비밀번호 변경`을 선택할 수 있습니다.
